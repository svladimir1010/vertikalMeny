Нужно сверстать этот макет. CSS фреймворки и библиотеки использовать можно, но в разумных пределах, это значит что не стоит подключать весь bootstrap, если он будет использоваться только для того чтоб сбросить стили).

Переключение пунктов слева нужно реализовать с помощью JavaScript (jQuery). При клике на другой пункт меню, он должен активироваться, а содержание всего сиренового блока должно меняться.

Горячо приветствуется использование препроцессоров для CSS. Не забудьте приложить исходник.

Рассчитывать на ширину экрана 1000 пикселей. Он не должен быть “пиксель в пиксель”, но соотношений сторон и пропорций лучше придерживаться.

При активации пункта меню смена содержания сиреневого блока, желательно, должа происходить с анимацией.
